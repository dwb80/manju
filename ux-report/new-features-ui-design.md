## 📐 页面设计规范

### 通用布局结构

```
页面整体结构（垂直布局）
├── Header（粘性固定）
│   ├── 导航区（返回、面包屑、全局操作）
│   ├── 标题区（主标题 + 关键指标）
│   └── 工具栏（搜索、筛选、批量操作）
│
├── Section（主体内容）
│   ├── 概览区（统计卡片）
│   ├── 列表区（数据表格/卡片列表）
│   └── 详情区（侧边抽屉/模态框）
│
└── Footer（底部信息）
    ├── 左侧：实时状态
    └── 右侧：最后更新时间
```

### 响应式断点

| 断点名称 | 屏幕宽度 | 布局特点 |
|----------|----------|----------|
| Mobile | <768px | 垂直布局，隐藏次要信息 |
| Tablet | 768px-1024px | 混合布局，保留核心信息 |
| Desktop | >1024px | 完整横向布局，显示所有功能 |

## 🔍 模块一：审核队列管理页面设计

### 页面功能概述

审核队列管理页面用于管理AI生成内容的审核流程，确保内容质量符合发布标准。

### 核心功能

1. **待审核内容列表**
   - 显示待审核的图片/视频内容
   - 支持缩略图预览
   - 显示内容来源和生成参数

2. **审核操作面板**

   - 批注和反馈
   - 质量评分

3. **审核统计仪表盘**

### 交互设计

#### 快捷审核流程

1. 用户点击内容行进入详情抽屉
2. 查看大图预览和详细信息
3. 拖动评分滑块或点击星星评分
4. 输入审核意见（可选）
5. 点击"通过"或"拒绝"按钮
6. 系统自动加载下一个待审核内容

#### 批量审核流程

1. 勾选多个待审核内容
2. 点击"批量通过"或"批量拒绝"
3. 系统显示批量操作确认对话框
4. 确认后批量更新状态
