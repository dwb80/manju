# 资产库使用说明

资产库是 AI 漫剧项目复用素材的地方。它不是普通文件列表，而是把图片、视频、角色、场景、风格、提示词、项目资料和分镜底图统一管理起来，方便后续生成图片、图生视频和剪辑交付。

## 入口

在左侧选择一个具体项目后，进入主区域的 `项目工作台`，打开 `资产库` 页签。

资产必须归属到某个项目。如果当前选择的是“全部项目”或“不使用项目”，资产库不会作为具体项目资产展示。

## 支持的资产类型

- 图片资产：生成图、上传图、参考图。
- 视频资产：生成视频、上传视频或外部视频链接。
- 角色资产：角色卡，用于保存人物参考图、角色描述、外观标签和固定提示词。
- 场景资产：场景卡，用于保存地点参考图、空间描述、时间气氛、光照和场景固定提示词。
- 风格资产：画风模板，用于保存风格提示词、参考图、色调、光影和镜头风格。
- 提示词资产：可复用 prompt 模板。
- 项目资产：剧本资料、世界观、设定文档等项目级素材。
- 分镜底图：每个镜头的底图或关键画面，可直接作为图生视频参考图。

## 新建资产

常用字段：

- 名称：资产在列表中展示的标题。
- 设定词 / 生成提示词：复用时会自动带入输入框。
- 图片 / 参考图 URL：图片、角色、场景、风格、分镜底图常用。
- 视频 URL：视频资产常用。
- 文件夹：手工分类，例如 `角色/主角`、`场景/长街`、`第一集/分镜`。
- 标签：用逗号分隔，例如 `主角, 定妆, 古风`。
- 分辨率：图片或视频规格，例如 `1080P`。
- 时长：视频资产时长，例如 `5秒`。
- 角色特征：角色资产使用，例如 `长发, 白衣, 少年感`。
- 风格关键词：风格或场景资产使用，例如 `雨夜, 暖光, 电影感`。

保存后，资产记录会写入 SQLite 的 `project_assets` 表。

## 搜索和筛选

资产库顶部支持：

- 关键词搜索：按名称、提示词、备注、文件夹和标签搜索。
- 类型筛选：按图片、视频、角色、场景、风格、提示词等筛选。
- 标签筛选：只显示包含指定标签的资产。
- 收藏筛选：只显示已收藏资产。

## 常用操作

- 收藏 / 取消收藏：点击资产卡片右侧星标。
- 删除：删除资产库记录。
- 复用：把资产提示词追加到输入框，并把参考图带入附件区。
- 用于视频：把图片、角色图、场景图或分镜底图作为视频参考图。
- 复制提示词：复制资产保存的 prompt。
- 设为参考图：把图片 URL 放入图片生成流程，适合二次编辑。

## 角色资产建议

角色资产相当于角色卡，建议这样填写：

- 名称：角色名。
- 参考图 URL：角色主参考图。
- 设定词 / 生成提示词：角色固定描述。
- 角色特征：发型、脸型、服装、体态、标志物。
- 标签：主角、反派、第一集等项目标签。

复用角色资产时，系统会把角色 prompt 和参考图带入生成输入区，用于提升同一人物的一致性。

## 场景资产建议

场景资产相当于场景卡，建议这样填写：

- 名称：场景名，例如 `雨夜长街`。
- 参考图 URL：场景参考图或场景底图。
- 设定词 / 生成提示词：空间结构、时代背景、建筑材质、天气、时间。
- 风格关键词：光照、色调、镜头氛围，例如 `湿润石板路, 暖色窗光, 低机位`。
- 标签：室外、夜景、第一集、常驻场景等。

复用场景资产时，系统会把场景描述和参考图带入生成输入区，减少每条分镜重复描述场景的成本。

## 分镜底图建议

分镜底图适合保存每个镜头的关键画面，例如：

- 第一集第 1 镜开场底图。
- 角色进入场景前的站位图。
- 图生视频前需要锁定的构图参考。

在资产卡片上点击 `用于视频`，系统会把底图放入视频生成附件区，并切换到视频生成页面。
